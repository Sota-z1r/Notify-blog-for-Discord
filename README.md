# プロジェクト名
  Notify-blog-for-Discord
## 概要

このプロジェクトは、特定のブログサイトの更新をDiscordに通知するためのものです。

## インストール


1. [仮想環境の作成]
   python3 -m venv venv
2. [仮想環境の有効化]
   source venv/bin/activate 
3. [プロジェクトの依存関係をインストールするコマンド]
   pip install -r requairement.txt
4. [DiscordのWebhookURLを設定]
   WEBHOOK_URL = "YOUR_WEBHOOK_URL_HERE"


## 使い方

1. [プロジェクトを実行するコマンド]
   caffeinate -i python discord_notify.py
